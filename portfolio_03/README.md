## ✨ Features

- 🌐 **3D Hero Scene** — Rotating data globe with wireframe layers, orbit rings, surface nodes, and floating 3D polyhedra (Three.js)
- 🔮 **3D About Sphere** — Animated wireframe sphere with triple orbit rings and glowing orbiting dots
- 🃏 **3D Tilt Cards** — Mouse-tracked perspective tilt with glare/shine effect on all cards
- 🖱️ **Custom Cursor** — Wine-red cursor with smooth trailing ring
- 📜 **Scroll Animations** — Fade-up entrance animations via IntersectionObserver
- 📊 **Animated Skill Bars** — Progress bars that animate on scroll
- 🔢 **Counter Animations** — Stats count up when scrolled into view
- 🎯 **Parallax** — Hero content shifts with scroll depth
- 🧭 **Scroll-Spy Nav** — Active link highlights as you scroll
- 📱 **Fully Responsive** — Mobile-first with animated hamburger menu

## 🚀 Getting Started

### Local Preview
Simply open `index.html` in any modern browser. No build step required.

```bash
# If you have VS Code with Live Server
# Right-click index.html → Open with Live Server

# Or via Python (optional)
python -m http.server 8080
# Then visit: http://localhost:8080
```

### Deploy to GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from branch → main → / (root)**
4. Your portfolio will be live at:
   ```
   https://<your-username>.github.io/<repo-name>/
   ```
